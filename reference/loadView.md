# Help function to load the view angle for the RGL 3D plot from a file or matrix

Help function to load the view angle for the RGL 3D plot from a file or
matrix

## Usage

``` r
loadView(
  fname = "view.RData",
  v = NULL,
  clear = TRUE,
  close = FALSE,
  zoom = 1,
  ...
)
```

## Arguments

- fname:

  The file name of the view.

- v:

  The view matrix.

- clear:

  Call
  [`rgl::clear3d()`](https://dmurdoch.github.io/rgl/dev/reference/scene.html).

- close:

  Call
  [`rgl::close3d()`](https://dmurdoch.github.io/rgl/dev/reference/open3d.html).

- zoom:

  Zoom level.

- ...:

  Additional parameters passed to
  [`rgl::view3d()`](https://dmurdoch.github.io/rgl/dev/reference/viewpoint.html).
